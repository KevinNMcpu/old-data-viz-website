Weather Visualization - Latitudes

Welcome to my project's web page. The purpose of this project was to analyze and visualize the effect latitude has on four weather metrics- Temperature, Humidity, Cloudiness and Wind Speed

I randomly picked 500 locations on the planet by selecting randomly generating longitude and latitude, then used the very helpful Python library citiypy. Some data wrangling was then used to weed out duplicates as needed. Weather data was pulled from OpenWeather. Matplotlib was then used to do the visualizations in a Jupyter Notebook.
